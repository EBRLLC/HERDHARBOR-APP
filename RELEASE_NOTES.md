# HerdHarbor Pre-Alpha v0.2.1

This release focuses on tester feedback about pedigree difficulty and overall interface clutter.

## Highlights

- Added a cleaner, grouped navigation layout
- Added persistent light and dark modes
- Added a collapsible desktop sidebar
- Rebuilt pedigree creation as a five-step guided workflow
- Added pedigree drafts that can be saved and resumed
- Added drag-and-drop source uploads
- Added automatic image compression for large pedigree photos
- Added a printable three-generation sale pedigree
- Improved mobile usability throughout the pedigree workflow

## Guided pedigree builder

The pedigree builder now separates the process into:

1. Animal and source
2. Parents
3. Grandparents
4. Great-grandparents
5. Review and save

Ancestors may be linked to existing HerdHarbor animals, entered as ancestor-only records, or left unknown.

## Printable sale pedigree

Users can now generate a clean pedigree for sale or transfer. The printable record includes ancestry, registration details, buyer and seller information, sale details, notes, and signature lines.

## Appearance controls

Users can switch between light mode and dark mode from the app header. Settings also includes Light, Dark, and Use System options. Printed pedigrees remain white regardless of the selected theme.

## Important pre-alpha limitations

- Records and uploaded documents remain browser-only
- No cloud sync or secure user accounts yet
- Automatic pedigree OCR is not included
- PDF uploads are limited to 1.25 MB
- Users should export backups regularly

Please report bugs and confusing workflows through the in-app **Send Feedback** button.

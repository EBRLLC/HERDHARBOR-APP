# Pre-Alpha Data Notice HerdHarbor Pre-Alpha v0.1 stores records only in the browser on the device being used. It does not transmit livestock records to HerdHarbor, create a server account, or provide cloud backup. Users are responsible for exporting backups. Browser resets, device loss, private browsing, storage cleanup, or application changes may permanently erase local records. Do not enter payment information, government identifiers, passwords, protected medical data, or other highly sensitive personal information into this pre-alpha build.
